Tech

This website was made using Python, PostgreSQL, HTML and CSS.

Main Python frameworks/libraries used: FastAPI, Jinja, SQLAlchemy and Pydantic.

The list of matches and their odds are updated every 24 hours using the-odds-api.com, this means that the probabilities and odds presented are usually somewhat out of date.

Upcoming features

Password reset and page for account details.

Pages with more extensive information for each match and more frequent updates.

The website has fully functioning Oauth2 authorization which currently serves no practical use. In the future users will have the ability to store and analyze bets (whether real or simulated) and be able compare odds from different bookmakers.
